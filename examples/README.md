# 📚 Ejemplos

Esta carpeta contendrá ejemplos de uso de los skills y outputs generados.

## Estructura

Cada skill puede tener su propia subcarpeta con ejemplos:

```
examples/
├── documentation/
│   ├── sample-project/
│   │   ├── project-documentation.html
│   │   ├── documentation-styles.css
│   │   └── documentation-script.js
│   └── README.md
└── testing/
    └── README.md
```

## Cómo Agregar Ejemplos

1. Crea una subcarpeta con el nombre del skill
2. Agrega archivos de ejemplo reales
3. Documenta el contexto en un README.md
