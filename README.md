## 🚀 Cómo Usar

### Método 1: Copiar y Pegar (Básico)
1. Navega al skill que necesitas en la carpeta `skills/`
2. Copia todo el contenido del archivo `.md`
3. Pégalo en GitHub Copilot Chat (VS Code o GitHub.com)
4. Escribe tu solicitud específica

### Método 2: Snippets de VS Code (Rápido) ⚡
1. Instala los snippets: `npm run generate-snippets`
2. En Copilot Chat escribe: `!skill-doc` + TAB
3. El skill se carga automáticamente
4. Escribe tu solicitud

### Método 3: GitHub.com (Análisis Profundo)
1. Usa el skill directamente en github.com/rafael924/[tu-proyecto]
2. Copilot tiene acceso completo al repositorio
3. Mejor para skills que requieren análisis exhaustivo
