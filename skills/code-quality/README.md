# 📋 Skills de Code Quality

Esta categoría contendrá skills relacionados con la mejora de la calidad del código.

## Skills Planeados

- **Code Reviewer**: Revisión experta de código con sugerencias de mejores prácticas
- **Refactoring Assistant**: Asistente para refactorización basado en patrones de diseño
- **Security Auditor**: Identificación de vulnerabilidades y problemas de seguridad

## Contribuir

Si tienes ideas para skills en esta categoría, usa la [plantilla de skill](../../templates/skill-template.md) y crea un Pull Request.
