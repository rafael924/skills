# ⚙️ Skills de Automation

Esta categoría contendrá skills relacionados con automatización de tareas de desarrollo.

## Skills Planeados

- **CI/CD Helper**: Asistencia para configurar y optimizar pipelines
- **Deployment Assistant**: Automatización de procesos de despliegue
- **Git Workflow Optimizer**: Optimización de flujos de trabajo con Git
